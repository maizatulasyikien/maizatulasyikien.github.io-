Engineering Sustainable Mobile Systems MakanJimat

MakanJimat is a sustainable food rescue application developed independently to bridge the gap between restaurant surplus inventory management and responsible consumer consumption. Built using modern Android architecture, the app integrates local data persistence, real-time cloud synchronization, and hardware GPS sensors to deliver a seamless, eco-friendly food ecosystem on a single mobile device. It is designed to empower users to rescue high-quality surplus meals at discounted prices while simplifying the transition toward a circular economy and reducing the environmental footprint of the local food industry.

Project 1 Milestone Navigation Core MVP

MAKANJIMAT (Architectural Framework)

Problem Statement

The modern food industry generates massive amounts of high-quality surplus food that is discarded daily due to inefficient redistribution channels and a lack of consumer awareness. This cycle creates a critical environmental and social burden, as valuable resources are wasted while carbon emissions from food disposal continue to accelerate global climate change.

Lessons Learned

I learned how to scale a multi-screen application for MAKANJIMAT by establishing a robust navigation framework using NavHost to manage complex state transitions across five distinct screens. This phase deepened my understanding of state hoisting through a centralized ViewModel, enabling me to share 'food rescue' data seamlessly between UI components without the need for persistent storage. Additionally, I gained hands-on experience in implementing Material 3 design principles and custom styling to ensure a consistent, professional user experience that clearly communicates the impact of SDG 12 to the user.

Project 2 Milestone API & Cloud Suite

MAKANJIMAT (Ecosystem Deployment)

Problem Statement

Effective food waste reduction requires real-time visibility of surplus inventory and community-wide data synchronization, which traditional static food applications cannot provide. MakanJimat solves this by integrating web APIs for live sustainability news, hardware GPS sensors for immediate store discovery, and secure cloud databases to guarantee that surplus food rescue opportunities are identified instantly and environmental impacts are recorded accurately for the community.

Lessons Learned

Developing MAKANJIMAT taught me how to architect a complex Android ecosystem that handles concurrent asynchronous tasks using Kotlin Coroutines and Retrofit for real-time sustainability data fetching. I gained invaluable hands-on experience in utilizing the Android hardware sensor framework for GPS-based store discovery and configuring cloud infrastructures like Firebase Firestore to bridge local Room database workflows smoothly with remote backends. This journey solidified my understanding of modern Android architecture and demonstrated how integrating diverse technical pillars—API, Sensors, and Persistence—can create a powerful, data-driven solution for achieving SDG 12.
